# linksphere-url-shortener
LinkSphere is a URL management system that converts long URLs into short, unique links for easy sharing. It simplifies link handling while ensuring efficient redirection and better usability.

## Features
- URL shortening  
- Unique link generation  
- Simple interface  

## Tech Stack
- Python  
- Streamlit  

## Project Structure
app.py
requirements.txt
README.md

## Installation
pip install -r requirements.txt

## Run the Project
streamlit run app.py

## How It Works
- User enters a URL  
- System generates a short hash  
- Displays shortened link
